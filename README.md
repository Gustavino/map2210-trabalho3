# Universidade de São Paulo - Instituto de Matemática e Estatística (IME-USP)

## MAP 2210 - Aplicações de Álgebra Linear

### Trabalho Computacional III

 * a) Escolha 10 matrizes simétricas positivas definidas do repositório https://sparse.tamu.edu/.  
 * b) Implemente o [método da potência](https://en.wikipedia.org/wiki/Power_iteration) e encontre o raio espectral de cada matriz.  
 * c) Utilize o [método da potência inverso](https://en.wikipedia.org/wiki/Inverse_iteration) e repita o estudo anterior.  
 * d) Desenvolva e aplique o [algoritmo QR](https://en.wikipedia.org/wiki/QR_algorithm) para encontrar o espectro completo de cada matriz.
 * e) Discuta os resultados.
